# Smart Interview Analyser 🎯

## 📌 Project Overview

The **Smart Interview Analyser** is an AI-powered application designed to help users improve their interview skills through real-time analysis of their responses. It evaluates answers based on predefined keywords, sentiment, and structure, providing constructive feedback to enhance performance.

This project is especially useful for students and job seekers preparing for technical and HR interviews.

---

## 🚀 Key Features

*  **AI-Based Answer Evaluation** using Natural Language Processing (NLP)
*  **Predefined Interview Questions** (HR-focused)
*  **Sentiment Analysis** using TextBlob
*  **Keyword Matching** to assess relevance of answers
*  **Real-Time Feedback System**
*  **Performance Scoring**
*  **Interactive UI using Streamlit**

---

## 🛠️ Technologies Used

* **Python**
* **Streamlit** – for web interface
* **TextBlob** – for sentiment analysis
* **NLTK** – for text processing
* **Machine Learning Concepts (Basic NLP)**

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/smart-interview-analyser.git
cd smart-interview-analyser
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```

## 🧪 How It Works

1. User selects or is presented with an interview question
2. User types their answer into the interface
3. The system:

   * Performs **sentiment analysis**
   * Checks **keyword relevance**
   * Evaluates **response quality**
4. Displays:

   * Feedback
   * Score
   * Suggestions for improvement
